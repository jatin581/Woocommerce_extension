# Spiderman Plugin

## Description
Spiderman Plugin is a WordPress plugin that adds a top-level menu item with a submenu page in the WordPress admin sidebar. It also makes an API call to retrieve data and displays it in an iframe.

## Features
- Adds a top-level menu item named "Spiderman Plugin" with a submenu page in the WordPress admin sidebar.
- Makes an API call to retrieve data.
- Displays the API response content in an iframe on the submenu page.

## Installation
1. Download the plugin ZIP file from the [releases page](https://github.com/your-username/spiderman-plugin/releases).
2. Log in to your WordPress admin dashboard.
3. Navigate to Plugins > Add New.
4. Click on the "Upload Plugin" button.
5. Choose the ZIP file you downloaded and click "Install Now."
6. After installation, activate the plugin.

## Usage
1. After activation, you'll see a new menu item named "Spiderman Plugin" in the WordPress admin sidebar.
2. Click on the menu item to access the submenu page.
3. The submenu page will display content retrieved from an API call, embedded in an iframe.

## Configuration
- The plugin does not require any additional configuration. However, you can modify the API endpoint and parameters in the code if needed.
